# Tkinter로 만든 커넥트 4 게임

이 스크립트는 파이썬에서 Tkinter를 사용하여 간단한 커넥트 4 게임을 구현합니다.

## 전제 조건

시스템에 파이썬이 설치되어 있는지 확인하십시오. Tkinter는 표준 라이브러리에 포함되어 있으므로 추가 모듈이 필요하지 않습니다.

## 스크립트 실행 방법

1. 프로젝트 디렉토리로 이동합니다.

    ```bash
    cd Connect-Four
    ```

2. 스크립트를 실행합니다.

    ```bash
    python main.py
    ```

3. 게임 창이 열리고 하단의 버튼을 클릭하여 말을 놓아 게임을 시작할 수 있습니다.
