# 간단한 HTTP 서버

## 🛠️ 설명
파이썬 소켓을 사용하여 작성된 간단한 HTTP 서버입니다.

## ⚙️ 사용된 언어 또는 프레임워크
이 프로젝트는 파이썬 3.10으로 작성되었으며 파이썬 표준 라이브러리 외에는 다른 종속성이 없습니다.

## 🌟 실행 방법
`python mhttp.py <원하는 폴더>`를 실행하기만 하면 됩니다. 여기서 `<원하는 폴더>`는 제공해야 하는 폴더입니다. 현재 폴더를 제공하려면 `python mhttp.py`를 실행하기만 하면 됩니다.
