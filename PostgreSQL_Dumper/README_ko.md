# PostgreSQL 데이터베이스 덤퍼

## ⭐ 설명
파이썬에서 PostgreSQL 데이터베이스의 덤프를 생성하는 간단한 스크립트입니다.

## ⚙️ 실행 방법
***요구 사항을 설치하고 터미널을 통해서만 실행하십시오.***

PyCharm 또는 Fleet에서 다음과 같은 오류가 발생할 수 있습니다.

`termios.error: (25, '장치에 부적절한 ioctl')`

이는 키 수신을 지원하지 않기 때문에 발생합니다.
