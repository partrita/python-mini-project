## 🌟 실행 방법
<!--아래 줄을 삭제하고 원하는 내용을 추가하십시오-->
```bash
python3 hanoi.py

맨 위 숫자를 가져올 열(1, 2 또는 3)을 선택한 다음 해당 숫자를 이동할 열을 선택합니다. 숫자는 항상 그 아래 숫자보다 작아야 합니다.
목표: 전체 스택을 다른 열로 이동합니다.

1 |   |
2 |   |
3 |   |

열? 1
이동할 곳? 2

  |   |
2 |   |
3 | 1 |
```
