## 🛠️ <b> 설명 </b>

## ⚙️ <b>사용된 언어 또는 프레임워크
다음 명령을 실행합니다.
```sh
$ python -m pip install --upgrade pip
$ python -m pip install opencv-python
```

```sh
$ python lanes.py
```
