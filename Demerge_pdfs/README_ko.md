# PDF 분할

## 🛠️ 설명

큰 PDF 파일을 변경하지 않고 여러 다른 크기의 PDF 파일로 변환하는 파이썬 프로그램입니다.

## ⚙️ 사용된 언어 또는 프레임워크
PyPDF2, 명령줄에 `pip3 install PyPDF2`를 입력하여 설치할 수 있습니다.

## 🌟 실행 방법
스크립트를 실행하는 것은 정말 간단합니다! 이 스크립트가 있는 폴더에서 터미널을 열고 다음 명령을 실행하기만 하면 됩니다.
```sh
python demerging_pdf.py
```
