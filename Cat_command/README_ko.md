# 고양이 명령어

## 🛠️ 설명
파이썬을 사용하여 구현된 리눅스의 "cat" 명령어.
명령어 인수에 지정된 파일을 열고 콘솔에 인쇄합니다.

## 요구 사항
- argparse
- pathlib

## 🌟 실행 방법
리눅스에서는 다음을 사용할 수 있습니다.
``` ./cat.py [경로] ```
또는
```python ./cat.py [경로]```
예시
```.cat.py ./test_cat.txt```
