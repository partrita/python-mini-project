# 스크립트 제목
여러 이미지를 PDF로 변환기

## 🛠️ 설명
<!--아래 줄을 삭제하고 원하는 내용을 추가하십시오-->
폴더에 있는 여러 이미지를 하나의 PDF로 변환합니다.

## ⚙️ 사용된 언어 또는 프레임워크
<!--아래 줄을 삭제하고 원하는 내용을 추가하십시오-->

## 🌟 실행 방법
<!--아래 줄을 삭제하고 원하는 내용을 추가하십시오-->

명령: python3 images_to_pdf.py
