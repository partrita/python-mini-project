# 체스 게임

## 🛠️ 설명
친구와 함께 즐길 수 있는 간단한 체스 게임입니다.

## ⚙️ 사용된 언어 또는 프레임워크
이 게임은 파이썬 프로그래밍 언어로 만들어졌습니다.
모듈 : pygame

## 🌟 실행 방법스크립트 실행 방법
이 게임을 실행하는 것은 매우 간단합니다.

```sh
pip install -r requirements.txt
```

```sh
python ChessGame.py
```
